# Excel to Pipe | Conversor Premium

Conversor premium de archivos Excel a formato delimitado por tubería (|). Desarrollado como una Progressive Web App (PWA) rápida, segura y con funcionamiento offline.

## 🚀 Características
- **Conversión Instantánea**: Transformación rápida de archivos `.xlsx` a CSV delimitado por `|`.
- **Funcionamiento Offline**: Gracias a su tecnología PWA, puedes usarla sin conexión a internet.
- **Privacidad Total**: El procesamiento se realiza localmente en tu navegador; tus archivos nunca se suben a un servidor.
- **Interfaz Moderna**: Diseño fluido, oscuro y minimalista.

## 🛠️ Instalación
Al ser una PWA, puedes "instalar" la aplicación en tu escritorio o móvil directamente desde el navegador (Chrome, Edge, etc.) haciendo clic en el icono de instalación en la barra de direcciones.
